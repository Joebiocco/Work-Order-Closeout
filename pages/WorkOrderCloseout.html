How to Use Work Order Closeout
1
Enter Work Order Details
Fill in the work order, structure, route, direction, milepost, dates, and priority before adding documentation pages. These fields become the header information in the PDF.
2
Add Photos and Notes
Use Camera or Gallery on each photo card, then enter the location and description. Add pages when the closeout needs more photos.
Camera Gallery Add Page
3
Save the Editable Session
Tap Save Session to download the HTML session file. That saved HTML becomes the editable copy of the work order, so keep it with the job files if anyone may need to change text, add photos, or regenerate the PDF later.
4
Only Five Recent Sessions
Recent is temporary device storage and only keeps the latest five sessions. Older sessions can be replaced automatically, so do not rely on Recent as the permanent record. Always keep the downloaded HTML session file.
5
Save Both Files
Tap Export PDF for the final closeout document, and also keep the HTML session file. The PDF is basically a flat image for sharing/records; the HTML file is the editor that lets you reopen, revise, and export again.
